# CentOs 7 Step-By-Step

#### Fri, 31st July, 2020

## 1. Update & upgrade
```
~ sudo yum -y update && sudo yum -y upgrade
```

## 2. Configure common settings
```
# install dnf
~ sudo yum -y install dnf

# Logout and login 
~ gnome-session-quit

# install vim-enhanced
```

## 2. Change to zsh shell
```
~ sudo chsh -s /usr/bin/zsh [user]
```
